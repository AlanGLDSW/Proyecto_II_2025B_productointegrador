Sistema para la gestión administrativa del Residencial Los Robles

Desarrollo ágil de una plataforma administrativa

Este proyecto nace de la necesidad urgente de modernizar la administración del Residencial Los Robles, una comunidad que enfrentaba desafíos críticos en gestión, comunicación y transparencia.


Este proyecto consiste en una plataforma web diseñada para optimizar la administración y comunicación del Residencial Los Robles. Desarrollado mediante metodologías ágiles, el sistema centraliza los procesos administrativos, facilitando la interacción entre residentes y administración.


Desarrollé una plataforma web integral que centraliza y digitaliza todos los procesos administrativos, implementando módulos clave como: sistema de pagos en línea para cuotas de mantenimiento, comunicación masiva en tiempo real, votaciones electrónicas participativas, botón de emergencias con geolocalización, y transparencia financiera completa. Aplicando metodología Scrum a lo largo de 8 sprints planificados, logré entregar valor incremental priorizando las funcionalidades más críticas para los residentes, demostrando cómo el desarrollo ágil puede transformar digitalmente comunidades residenciales hacia una gestión más eficiente, transparente y centrada en el usuario.

Funcionalidades principales:

  1. Autenticación segura de residentes con registro y recuperación de contraseñas
  2. Dashboard interactivo con acceso rápido a servicios principales
  3. Sistema de pagos en línea para cuotas de mantenimiento
  4. Módulo de emergencias con geolocalización
  5. Panel administrativo para gestión de residentes y notificaciones
  6. Sistema de votaciones electrónicas seguras

Tecnologías implementadas:

  1. Frontend: HTML5, CSS3, JavaScript y React.js
  2. Backend: Firebase (base de datos y autentificación segura)
  3. Despliegue: GitHub.
  4. Metodología: Scrum.
